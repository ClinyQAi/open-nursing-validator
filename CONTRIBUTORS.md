## How to Contribute

### For Clinical Experts (Nurses, Healthcare Professionals)
1. Review the [Clinical Expansion Roadmap](.spec-kit/specs/06-clinical-expansion-roadmap.md)
2. Create a [Clinical Profile Request](https://github.com/ClinyQAi/nhs-unified-nursing-validator/issues/new?template=clinical_profile_request.yml)
3. We'll work with you to implement and validate the profile
4. Your name will be credited in the spec, code, and this file

### For Developers
1. See [CONTRIBUTING.md](CONTRIBUTING.md) for technical guidelines
2. Pick an issue labeled `good-first-issue` or `clinical-profile`
3. Follow the Spec-Driven Development workflow

## Community Values

- **Clinical Accuracy First**: Validation rules must reflect real nursing practice
- **Person-Centred**: Technology serves care, not the other way around
- **Open Source**: Knowledge should be freely shared for the common good
- **Attribution**: We celebrate and credit all contributions
